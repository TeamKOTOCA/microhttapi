# pve02-macrohttp
古い非力なルーターに乗せるための超小型httpサーバーソフトウェア

## 初期設定
makeした実行ファイルと同じフォルダーに/wwwというフォルダーを作成してください。

エラーページは共通スタイルとして/www/errors.cssを参照します。エラーページ関連のデザインはここに記述してください。
規定のデザイン例はリポジトリ内の/www/errors.cssに配置してあります。

## 認証機能
コードのAUTH_KEYのところに任意のキーを記述してください。

アクセス形式は
http://example.com/index.html?key={指定したキー}
です。
